## Cómo agregar un proyecto nuevo

1. Crear una carpeta nueva en la raíz del repo, `nombre-del-proyecto/`.
2. Poner ahí el/los archivo(s) HTML, con `index.html` como página principal de esa carpeta.
3. Commitear y hacer push a `main` — GitHub Pages lo publica automáticamente en `https://santiagoolivos.github.io/public-pages/nombre-del-proyecto/`.
4. Sumar el link en la lista de arriba.
